\chapter{Code Availability}
\label{appendix:code}

To ensure transparency and reproducibility, the complete source code associated with this diploma thesis has been made publicly available on GitHub at:

\begin{center}
    \url{https://github.com/geokyr/diploma-thesis}
\end{center}

The repository includes:
\begin{itemize}
    \item Project documentation and usage instructions
    \item Dataset generation simulation with SUMO
    \item Machine learning research experiments
    \item Platform implementation (backend, predictor, drift, summarizer, frontend)
    \item Containerization and deployment artifacts (Dockerfiles, Docker Compose, uv)
\end{itemize}
