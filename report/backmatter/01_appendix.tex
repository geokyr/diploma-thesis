\chapter{Source Code Repository}

The complete source code for this diploma thesis is publicly available on GitHub:

\begin{center}
    \url{https://github.com/geokyr/diploma-thesis}
\end{center}

The repository includes:
\begin{itemize}
    \item Dataset generation simulation with SUMO and configuration files
    \item Machine learning research experiments
    \item Platform microservices (backend, frontend, predictor, drift, summarizer)
    \item Dockerfiles and Docker Compose
    \item Dependency management with uv
    \item Project documentation
\end{itemize}
