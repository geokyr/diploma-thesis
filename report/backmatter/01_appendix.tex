\chapter{Source Code Repository}
\label{appendix:github}

The complete source code for this thesis is publicly available on GitHub:

\begin{center}
    \url{https://github.com/geokyr/thesis}
\end{center}

The repository includes:
\begin{itemize}
    \item SUMO configuration files and datasets
    \item Dataset generation simulation with SUMO
    \item Machine learning research experiments
    \item Platform microservices
    \item Docker Compose configuration
    \item Dockerfiles for each microservice
    \item Dependency management with uv
    \item Documentation for the project
\end{itemize}
